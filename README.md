# Expander

Expander is a RuneLite plugin for self-imposed area-locked accounts.

Choose an origin tile, then expand the unlocked area one level at a time. The
plugin draws the unlocked boundary, fades locked areas, marks the origin tile,
tracks the number of unlocked tiles, and provides small notes fields for
objectives and backlog items.

## Features

- Set and persist an origin tile.
- Expand or reduce the unlocked area.
- Show the current expansion level and unlocked tile count.
- Draw a boundary around the unlocked area.
- Mark the origin tile.
- Add personal objective and backlog notes.
- Optional sounds for expanding, reducing, setting the origin, and leaving the
  unlocked area.
